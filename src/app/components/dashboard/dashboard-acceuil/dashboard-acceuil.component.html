Bienvenue sur l'interface administrateur !

Ici vous pourrez créer, éditer et supprimer :

- Les onglets de la barre de navigation
- Les pages
- Les articles de l'accueil

Veuillez utiliser la barre de navigation sur la gauche de votre écran pour commencer à éditer les différents éléments de votre site

Si vous souhaitez aller dans la base de données cliquer sur 'Base de données Google', sélectionnez 'SynergieAngular' puis aller dans l'onglet 'Firestore Database' situé à gauche de l'écran
